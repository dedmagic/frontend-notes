# План

Создадим объект:

```js
const obj = {
    a: 42,
    b: 96
}
```

Для этого объекта мы можем вызвать методы, например, `hasOwnProperty` или `toString`.

```js
console.log(obj.hasOwnProperty());
console.log(obj.toString());
```
